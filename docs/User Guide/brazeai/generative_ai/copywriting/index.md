# Generate copy

> Generate marketing copy for SMS, push, and Canvas messages using Operator. Describe a product or campaign, include a tone if needed, and Operator drafts copy you can review and insert directly into your message composer.

**Note:**


This feature is now part of [BrazeAI Operator™](https://www.braze.com/docs/user_guide/brazeai/operator/capabilities/), Braze's unified AI assistant. AI capabilities that were previously available as standalone assistants are now all accessible through Operator from wherever you're already working in the dashboard.




## How to access {#how-to-access}

In any message composer, open Operator to draft and refine marketing copy. Describe your product or campaign, and Operator returns copy you can review and insert.


For more information, including available tones, example prompts, and brand guideline integration, see [Generate copy in Operator capabilities](https://www.braze.com/docs/user_guide/brazeai/operator/capabilities/#generate-copy).
