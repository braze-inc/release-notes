## Feature overview

| Your goal | Recommended feature |
| --- | --- |
| Personalize message copy per user using their context | [Braze Agents](https://www.braze.com/docs/user_guide/brazeai/agents) ([Canvas](https://www.braze.com/docs/user_guide/messaging/canvas/canvas_components/agent_step) or [catalog](https://www.braze.com/docs/user_guide/brazeai/agents/deploying_agents#use-catalog-agents)); for one-off draft copy, [Generative AI](https://www.braze.com/docs/user_guide/brazeai/generative_ai) |
| Optimize which message content (such as subject, CTA) performs best over time | [Content Optimizer](https://www.braze.com/docs/user_guide/brazeai/content_optimizer) (email, push notifications, or SMS/MMS/RCS) or [Winning Variant](https://www.braze.com/docs/user_guide/messaging/ab_testing/optimizations) |
| Maximize a business metric (revenue, conversions) with 1:1 decisions across offers, channels, and timing | [Decisioning Studio](https://www.braze.com/docs/user_guide/brazeai/decisioning_studio) |
| Find users likely to churn or to perform an event | [Predictive Churn](https://www.braze.com/docs/user_guide/brazeai/predictive_suite/predictive_churn) or [Predictive Events](https://www.braze.com/docs/user_guide/brazeai/predictive_suite/predictive_events) |
| Recommend specific products from a catalog in messages | [Item recommendations](https://www.braze.com/docs/user_guide/brazeai/item_recommendations) |
| Send at the best time or the best channel per user | [Intelligent Timing](https://www.braze.com/docs/user_guide/brazeai/intelligence_suite/intelligent_timing), [Intelligent Channel](https://www.braze.com/docs/user_guide/brazeai/intelligence_suite/intelligent_channel) |
| Create or refine copy and creative directly in the dashboard | [Operator](https://www.braze.com/docs/user_guide/brazeai/operator) |
