# Localization

> Learn about localization and supported languages for the Braze SDK, so you can connect with your users across the globe. For guidance on setting up localized messages, refer to [Localization](https://www.braze.com/docs/user_guide/messaging/messaging_fundamentals/localization/) in our Messaging fundamentals section.

## About localization

In addition to English, Braze supports several languages for SDK messages displayed in your app. 

When a user's phone language is set to one of the supported languages, SDK messages that are included by default for the messaging channel will be translated to that language. For example, if your app displays a message for connectivity issues, it will be translated to user's chosen language.
